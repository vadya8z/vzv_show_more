# jQuery Plugin vzv_show_more
simple show more button
## USE:
```markdown
jQuery(document).ready(function($){
    //One block
    $('#content').vzv_show_more({
      controls_align: 'center',
      control_btn_tag: 'button',
    });
    //OR many blocks
    $('.vzv_show_more_content').vzv_show_more();
});
```
